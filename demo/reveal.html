how to roundtable

A Simple Demonstration

First

You're going to need something to present!

(I know, that seemed obvious to me too)

If you Don't Have a Reveal.js deck handy ...

Head over to http://rvl.io to start creating a presentation!

When you're done, hit export to get the code you need, and save it to a file on your computer.

Upload to Round table

Add a New Presentation in Round Table,
upload the file you created.

almost there ...

Now you have the presentation!

Start a presentation session with it.
You will get a QR code to point your presenter device at.

DONE!

Now you're presenting with Round Table.

From the presenter interface, you can show viewers a QR code
to let them connect and view your presentation
on their mobile device, and ask questions as you go.

YOU control the flow of questions, meaning
no one has to wait to ask,
and no one has to interrupt.

WANT MORE?

Round Table is the original title of the project,
developed during a Hackathon in Dec. 2012.

Round Table was created by Drew Strickland
and Annie MacMackin, two innovative and driven hackers.

This same technology will be used to create a platform
for use by presenters and educators.  Stay tuned.
